# FHIR Simple Demo Application
This project illustrates the use of InterSystems IRIS FHIR as a service.

## Demo
A live demo is available here: http://projecting.com.br/isc14/

## Features
* Dashboard Components
  * Charts
  * Scores
  * Lists
* Responsible Tables
  * Pagination
  * Sortable Fields
  * Searchable Fields
  * Reference Fields
* Responsive Forms
  * Related Resources
* Authorization
  * API key
  * OIDC

## Dependencies
* AngularJS
* Bootstrap
